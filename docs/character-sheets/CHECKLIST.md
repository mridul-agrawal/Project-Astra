# Character Lock-In Checklist

Everything that must be decided before a character sheet can be filled. Work top-down — earlier items unblock later ones.

- [ ] **Identity**
  - [ ] Name (Sanskrit / devanagari)
  - [ ] Name (romanized)
  - [ ] Name (IPA pronunciation)
  - [ ] Epithet / title
  - [ ] Mythological lineage (which figure they draw from)
  - [ ] Faction
  - [ ] Banner / house / sub-affiliation
  - [ ] Age
  - [ ] Gender
  - [ ] Species (Human / Asura / Deva / Yaksha / etc.)
  - [ ] Height
  - [ ] Voice direction (one descriptor)

- [ ] **Story Placement**
  - [ ] Role archetype (Lord / Jagen / Est / Pent / Villain / Deuteragonist / etc.)
  - [ ] Recruitment chapter
  - [ ] Recruitment method (auto / talk / visit / defeat / branch)
  - [ ] Availability window (start chapter → end chapter)
  - [ ] Lord flag (yes / no)

- [ ] **Combat Class & Movement**
  - [ ] Starting class
  - [ ] Movement type (Foot / Horse / Flying / Armor / etc.)
  - [ ] Promotion path(s)
  - [ ] Pancha Bhuta affinity (None / Akasha / Vayu / Agni / Jala / Prithvi)
  - [ ] Personality enum (CC-01)

- [ ] **Stats — Bases (Lv 1)**
  - [ ] HP
  - [ ] STR
  - [ ] MAG
  - [ ] SKL
  - [ ] SPD
  - [ ] LCK
  - [ ] DEF
  - [ ] RES
  - [ ] CON
  - [ ] MOV

- [ ] **Stats — Growth Rates (% per level)**
  - [ ] HP
  - [ ] STR
  - [ ] MAG
  - [ ] SKL
  - [ ] SPD
  - [ ] LCK
  - [ ] DEF
  - [ ] RES
  - [ ] CON
  - [ ] MOV

- [ ] **Stats — Cap Modifiers (signed delta on class caps)**
  - [ ] HP / STR / MAG / SKL / SPD / LCK / DEF / RES / CON / MOV

- [ ] **Weapon Ranks**
  - [ ] Sword
  - [ ] Lance
  - [ ] Axe
  - [ ] Bow
  - [ ] Dagger
  - [ ] Tome
  - [ ] Staff

- [ ] **Skills & Abilities**
  - [ ] Personal skill — Sanskrit name
  - [ ] Personal skill — English name
  - [ ] Personal skill — mechanic
  - [ ] Personal skill — lore tie
  - [ ] Class skills (list)
  - [ ] Signature ability — Sanskrit name
  - [ ] Signature ability — English name
  - [ ] Signature ability — mechanic
  - [ ] Signature ability — lore tie

- [ ] **Equipment**
  - [ ] Starting inventory
  - [ ] Personal weapon — name
  - [ ] Personal weapon — stats
  - [ ] Personal weapon — lore

- [ ] **Mechanical Role**
  - [ ] Niche on team (why bring them over alternatives)

- [ ] **Visual — Silhouette**
  - [ ] Silhouette one-liner (front-card hook)
  - [ ] Silhouette description (battle-sprite scale)

- [ ] **Visual — Features**
  - [ ] Skin tone & texture
  - [ ] Hair color
  - [ ] Hair style & length
  - [ ] Eye color, shape, default expression
  - [ ] Build & posture / default stance

- [ ] **Visual — Garments**
  - [ ] Headwear
  - [ ] Top
  - [ ] Bottom
  - [ ] Armor pieces
  - [ ] Accessories

- [ ] **Visual — Identity Marks**
  - [ ] Iconography / sigil / motif
  - [ ] Family crest / banner symbol
  - [ ] Tattoos / scars / distinguishing marks

- [ ] **Visual — Weapon & Mount**
  - [ ] Weapon visual (material, ornament, scale)
  - [ ] Mount visual (if mounted class)

- [ ] **Visual — Color Palette (5 hex slots)**
  - [ ] Primary
  - [ ] Secondary
  - [ ] Accent
  - [ ] Metal
  - [ ] Cloth

- [ ] **Visual — Portraits**
  - [ ] Portrait — full HP
  - [ ] Portrait — wounded (HP < 50%)
  - [ ] Portrait — critical (HP < 25%)
  - [ ] Portrait — deceased
  - [ ] Stressed overlay (USE-04)
  - [ ] Battle-sprite vs portrait simplification notes

- [ ] **Visual — Animation & Pose**
  - [ ] Idle expression
  - [ ] Critical-hit pose
  - [ ] Mood references (3–5)
  - [ ] Do's & don'ts for the artist

- [ ] **Personality**
  - [ ] One-paragraph hook
  - [ ] Background (2–3 paragraphs)
  - [ ] Personality traits (5 adjectives + supporting line each)
  - [ ] Beliefs & dharma alignment
  - [ ] Top 3 motivations
  - [ ] Fears & weaknesses
  - [ ] Speech style (formality, rhythm, signature phrases)
  - [ ] Internal conflict
  - [ ] Character arc summary
  - [ ] Foreshadowing hooks
  - [ ] Family tree
  - [ ] Key relationships (free-form)
  - [ ] Theme / leitmotif notes

- [ ] **Dialogue Library**
  - [ ] Recruit quote
  - [ ] Level-up quotes (4–6)
  - [ ] Critical-hit quotes (3–5)
  - [ ] Death quote / last words (2–4 lines if Lord)
  - [ ] Class-change quote
  - [ ] Idle barks
  - [ ] Battle conversation seeds vs key NPCs
  - [ ] One-line identity (War's Ledger death line)

- [ ] **Design Meta**
  - [ ] Design intent (why this character exists)
  - [ ] Story role
  - [ ] Accessibility / difficulty tier
  - [ ] Comparable references ("like Hector but with X")
  - [ ] Risks & pitfalls
  - [ ] Open questions
