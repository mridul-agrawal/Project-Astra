# Opening Cutscene — BG Art Prompts (for ChatGPT image generation)

Three background plates for the opening cutscene (Scene 02), one per dialogue block. Each `BG-*.md` file holds a single ready-to-paste prompt — open it, copy all, paste into ChatGPT.

**Tips for the best / most cohesive results:**
- Generate **BG-1 first**, then **attach BG-1 as a reference image** when generating BG-2 and BG-3 so palette and style match across the three.
- Ask for **landscape (16:9)** output (≈1920×1080).
- These produce a pixel-art-*styled* illustration — ideal as concept/placeholder art and as a reference to feed PixelLab later, **not** a final grid-perfect game asset. Fire/smoke/light are added in the engine, so the prompts deliberately ask for *environment only*.

**Files:**
- `BG-1_hunters-forest.md` — Block 1 (the hunt; dusk jungle)
- `BG-2_wrong-trail.md` — Block 2 (monstrous tracks + distant smoke)
- `BG-3_home-burning.md` — Block 3 (burning-village dialogue backdrop)
